# 🎓 교실 도우미

**텍스트 읽어주기 & 우리반 소음 측정기**

선생님과 학생들을 위한 스마트 교실 관리 도구입니다.

---

## ✨ 주요 기능

### 📢 텍스트 읽어주기 (TTS)
- 입력한 텍스트를 자동으로 음성으로 변환
- 6개 언어 지원 (한국어, 영어, 일본어, 중국어, 스페인어, 프랑스어)
- 읽기 속도 조절 (0.5배 ~ 2배)
- 음높이(피치) 조절 기능
- 음성 파일 MP3로 다운로드 가능
- 예시 문장 제공

### 📊 우리반 소음 측정기
- 실시간 교실 소음 측정 (데시벨)
- 시각적 경고 시스템 (조용함 🟢 / 보통 🟡 / 시끄러움 🔴)
- 현재/최대/평균 소음 통계
- 실시간 그래프로 소음 변화 추적
- 소음 기준 안내

## 🚀 빠른 시작 가이드 (비개발자용)

### 1️⃣ 준비물
- 컴퓨터 (Windows, Mac, Linux 모두 가능)
- 인터넷 연결
- 마이크 (소음 측정기 사용 시)

### 2️⃣ 프로그램 설치 (처음 한 번만)

#### Mac 사용자:
터미널을 열고 아래 명령어를 차례대로 입력하세요:

```bash
# 1. 이 폴더로 이동
cd /Users/jkey/Downloads/app

# 2. Python 설치 확인 (이미 설치되어 있을 수 있음)
python3 --version

# 3. 필요한 프로그램들 설치
pip3 install -r requirements.txt

# 4. 추가 설치 (Mac용)
brew install portaudio
```

#### Windows 사용자:
명령 프롬프트(CMD)를 열고:

```bash
# 1. 이 폴더로 이동
cd C:\Users\[사용자명]\Downloads\app

# 2. Python 설치 확인
python --version

# 3. 필요한 프로그램들 설치
pip install -r requirements.txt
```

### 3️⃣ 프로그램 실행

#### Mac:
```bash
streamlit run app.py
```

#### Windows:
```bash
streamlit run app.py
```

실행하면 자동으로 웹 브라우저가 열립니다!
(또는 `http://localhost:8501`을 브라우저에 입력)

## 📱 사용 방법

### 텍스트 읽어주기
1. 상단 탭에서 "📢 텍스트 읽어주기" 선택
2. 왼쪽 텍스트 상자에 읽을 내용 입력
3. 오른쪽에서 언어, 속도, 음높이 설정
4. "🔊 읽기 시작" 버튼 클릭
5. (선택) "📥 음성 파일 다운로드"로 MP3 저장 가능

### 소음 측정기
1. 상단 탭에서 "📊 우리반 소음 측정기" 선택
2. "🎙️ 측정 시작" 버튼 클릭
3. 브라우저에서 마이크 권한 허용
4. 실시간으로 교실 소음이 측정됩니다
5. "⏸️ 측정 중지"로 중단 가능

## 🌐 인터넷으로 배포하기 (무료)

### Streamlit Community Cloud 사용 (추천)

**5분이면 전 세계 어디서나 접속 가능한 웹사이트로 만들 수 있습니다!**

#### 1단계: GitHub 계정 만들기
1. https://github.com 접속
2. "Sign up" 클릭하여 무료 계정 생성
3. 이메일 인증 완료

#### 2단계: 코드 업로드
1. GitHub에 로그인
2. 오른쪽 상단 "+" 버튼 → "New repository" 클릭
3. Repository name: `classroom-helper` (원하는 이름)
4. Public 선택
5. "Create repository" 클릭
6. 생성된 페이지에서 "uploading an existing file" 클릭
7. `app.py`와 `requirements.txt` 파일을 드래그 앤 드롭
8. "Commit changes" 클릭

#### 3단계: Streamlit Cloud에 배포
1. https://streamlit.io/cloud 접속
2. "Sign up" → GitHub 계정으로 로그인
3. "New app" 버튼 클릭
4. Repository: 방금 만든 `classroom-helper` 선택
5. Main file path: `app.py`
6. "Deploy!" 버튼 클릭

**완료! 🎉**

5분 후 자동으로 URL이 생성됩니다 (예: `https://classroom-helper-abc123.streamlit.app`)
이 주소를 다른 선생님들과 공유하면 어디서나 사용 가능합니다!

## ⚠️ 주의사항

### 소음 측정기
- 마이크 권한이 필요합니다 (브라우저에서 허용해주세요)
- 측정값은 상대적인 수치입니다 (전문 측정기와 다를 수 있음)
- 환경에 따라 보정이 필요할 수 있습니다

### 음성 읽기
- 인터넷 연결이 필요합니다 (gTTS 사용)
- 긴 텍스트는 생성에 시간이 걸릴 수 있습니다

## 🆘 문제 해결

### "마이크에 접근할 수 없습니다"
- 브라우저 설정 → 개인정보 보호 → 마이크 권한 허용

### "모듈을 찾을 수 없습니다" 오류
```bash
pip3 install -r requirements.txt
```
다시 실행

### "포트가 이미 사용 중입니다" 오류
```bash
streamlit run app.py --server.port 8502
```
다른 포트로 실행

### 프로그램 종료 방법
터미널에서 `Ctrl + C` 누르기
